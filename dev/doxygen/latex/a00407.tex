This function converts a Voigt notation index to the corresponding matrix indices. The mapping is based on the assumption that the Voigt notation indices 0, 1, and 2 represent the diagonal components {\ttfamily 00}, {\ttfamily 11}, and {\ttfamily 22}, respectively. The remaining Voigt notation indices (3, 4, and 5) correspond to the off-\/diagonal components ({\ttfamily 12} and {\ttfamily 21}, {\ttfamily 02} and {\ttfamily 20}, {\ttfamily 01} and {\ttfamily 10}).

The function asserts that the input index is within the valid range for Voigt notation (0 to 5). \mbox{\Hypertarget{a00407_gadb08b6f9cc40847b70286dc314a23000}\label{a00407_gadb08b6f9cc40847b70286dc314a23000}}
