\doxysubsection{Detailed Description}
The equation for the load control method reads \[
f(\mathrm{D}\mathbf{D}, \mathrm{D} \lambda)=
\mathrm{D} \lambda -  \hat{s}, \] where $\mathrm{D}\mathbf{D}$ is the increment of the solution vector and $\mathrm{D} \lambda$ is the load factor increment. The scalar $\hat{s} $ defines the requested size of the step.
