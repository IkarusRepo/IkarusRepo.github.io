\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename PS$>$\newline
struct Ikarus\+::\+Materials\+::\+Deviatoric\+Invariants$<$ PS $>$}
The three invariants are \[ I_1 = \la_1^2 + \la_2^2 + \la_3^2, \] \[ I_2 = \la_1^2 \la_2^2 + \la_2^2 \la_3^2 +
\la_1^2 \la_3^2, \] \[ I_3 = \la_1^2 \la_2^2 \la_3^2. \]

The deviatoric invariants are then defined as \[ W_1 = I_1 I_3^{-1/3}, \] \[ W_2 = I_2 I_3^{-2/3}, \]
