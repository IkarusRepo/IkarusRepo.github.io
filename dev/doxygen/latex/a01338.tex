The \mbox{\hyperlink{a01338}{Load\+Control}} control routine increases the last parameter of a nonlinear operator and calls a nonlinear solver.

This class represents the \mbox{\hyperlink{a01338}{Load\+Control}} control routine. It increments the last parameter of a nonlinear operator and utilizes a nonlinear solver, such as Newton\textquotesingle{}s method, to solve the resulting system at each step.
