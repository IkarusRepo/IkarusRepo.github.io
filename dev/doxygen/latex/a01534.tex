\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename ST\+\_\+$>$\newline
struct Ikarus\+::\+Materials\+::\+Blatz\+Ko\+T$<$ ST\+\_\+ $>$}
The energy is computed as \[ \hat{\Psi}(\la_1, \la_2, \la_3) = \frac{\mu}{2}(
\frac{1}{\la_1^2} + \frac{1}{\la_2^2} + \frac{1}{\la_3^2}
+ 2\la_1 \la_2 \la_3 - 5). \]

\begin{DoxyRemark}{Remarks}
A special Blatz-\/\+Ko material model is implemented here. It assumes material parameters which corresponds to a Poisson\textquotesingle{}s ratio of 0.\+25.
\end{DoxyRemark}
