\doxysubsection{Detailed Description}
The \mbox{\hyperlink{a01596}{Hydrostatic\+Stress}} struct provides a function call operator to calculate hydrostatic stress. In 2D, this assumes a plane stress state. Furthermore the stress components are divided by 2 in 2D.
